# Macro Micro VR Service - Release Log
Release Logs: https://github.com/xavier150/MMVS/wiki/Release-logs

##  v0.4.5 Alpha (Vore Update)
    - You can now visit the full digestive system of Talas.
    - Now the crumb respawn after a delay.
    - New proximity light to help the player see around in the dark.
    - Random Preset button back in the customization menu.
    - New Render Options. (SSGI, Lumen, Path Tracing)
    - VSync and HDR supports.
    - VR player can now crouch in game while standing.
    - VR player Character can now be rescaled.
    - VR Hands can be rescaled. (Accessibility)
    - Better procedural animations. (Stomp, IK, Arms stretch)
    - Better VR Fullbody Animations.
    - Better UI interaction and light redesign.
    - New Game master previews. (Stomp area and Stomp target.)
    - Better optimization of the game in general.
    - Better and clean animations.

    - New cheat codes:
        - Ragdoll Modes.
        - Character config swap.
        - Cobble RGB.
        - Cobble Light.
        - ...

## Games Settings
    - Games settings:
        - New detail and descriptions for options in setting pages.
        - Options and texts reorganize.
        - New Video settings page:
            - You can choose the display target when using several monitor.
            - Moved Windows Mode option from Graphics settings.
            - You can use custom video resolution.
            - Support for VSync.
            - Moved Max FPS options from Graphics settings.
            - New Max Background FPS
            - Support for HDR (1000 Nits  and 2000 Nits).

        - Graphics settings:
            - Key for switch Render Method Removed (Move in Debug Menu > Rendering)
            - Use Path Tracing option.
            - Force Use Lumen option.
            - Use Screen Space Global Illumination (SSGI) option.
            - Preferred Reflection Method options. (Support Lumen reflections with static lighting)

        - Sound settings page rename to "Audio".

        - Ergonomic settings:
            - Proximity Light intensity.
            - Show Size option (Display the player size.)

        - VR settings:
            - Hands Scale (Used for rescale hand only.)

        - Control settings:
            - New panel for Generic Controller.

        - Pofile settings:
            - Digestive Light intensity.
            - New Sweat Factor option.
            - Options for show / hide strands.

        - New Self Censor setting Page:
            - Moved blood and gore options from Ergonomic settings.

## Localization
    - Localization strings updated and new Hungarian language.
    - Better support for Imperial Unit.

## Techs
    - Engine Update (Unreal Engine 5.2.1 -> Unreal Engine 5.4.4)
    
    - Character Refactoring:
        - Full refactoring about character assets managment.
        - Character scripts and Character constrcut scripts refactoring. (Now that work using config files.)
        - Character type (Macro/Micro/GM) is now seperate from the chracter profile (Talas/Cobble/...)
        - Blender Character generation scripts Refactoring (Rig, Procedural Skinning, Collision assets, ...).
        - Blender Character Rig refactoring. 
        - Updated character pipeline in Blender.
        - All animations reimport with the new rig changes.

    - Character physical and kinematics interactions:
        - Better physical interactions with characters Actors. (Mass pressure, hit and general physics interactions.)
        - Full refactoring of the force volumes. 
          (Used when the player is pushed by the water and now by in the digestive system.)
        - New wind support.

    - New skeletal dynamic mesh collision system:
        Hybrid system that use simple static shapes for physical collisions and per poly dynamic collisions for contact.
        Used for Talas Character and Talas Shoes.

    - New MMVS Addon used for related MMVS content in Blender.
        - Import / Export settings
        - Mass Import / Export from multi files.
        - Character and assets management.
        - Scene managements.
        - Backward Compatibility update button.
    
    - New Teleport Module:
        - Component and script used for seperate the Teleport game logic from the game character. (More modular)
        - Used for VR teleport. (Movement, Turn, Replace)
        - And general teleport.

    - New VR Module:
        - Component and script used for seperate the VR game logic from the game character. 
          (More modular and can be used on vehicles in the future)
        - New modular VR Player position scripts. 
          (Allows the VR player to move independently of the actor character. This make possible de crouch in VR.)  
        - New VR profiles:
            - Modular object with embedded code per profile.
              (The player change profile depending the context.)


    - Full recfactoring of the digestive system:
        - New techs for full dynamic render meshs.
        - New tech for full dynamic collision meshs.
        - New tech for animate dynamic meshs.
            - New Encode / Decode scripts to share animated data to vertex shader. (Animated data or computed animated vertex)
            - New in-editor modifiers to deform the digestive system meshs.
            - New in-editor rigs for animate the digestive system modifiers.
            - New control panel to see runtime animation in-editor

    - New digestive full dynamic collision system:
        Animated per poly collision that manage character push and mouvement update.
        With frame rate limitor and distance update for optimization.

    - New Slime Systems:
        - Generate and animate dynamic strands. 
          (Can be used for sweat, grimes, drool and more.)
        - New config based slime points spawner. (Used for character bodys or digestive environment.)
      
    - New Ragdoll and Phys Animation Controls:
        - You can activate Ragdoll and semi Ragdoll mode (Cheat only)
        - Character Physics can now interact with and on the world.
      
    - Integration of the Gameplay Ability System and Gameplay Tags:
        - Full recfactoring of the Health system.
        - Health scripts was fully recreated for using the Game Ability System.

    - New Game Properties System:
        - Used for manage referanced propertys from several places.
        - Update all UI widgets properties to use the new Game Properties System.
          (Widget and Properties will communicate but now the properties is not part of the widgets.)
        
    - New General Game Settings module:
        Used to manage general game settings with all player. (Use Gameplay Ability System)
        (Game damages, time dilation, fast travel, Cheat codes...)
        - Accessible in Game Master menu or Host New Game page.

    - Support for extreme scales.
        - New unit system to avoid loss of precision.
        - Support Dynamic Near Clip Plane

    - New Character Grab System:
        Modular and config based grab system.
        - Full refactoring of the VR grab to use the new Character Grab System.
        - Full refactoring of Talas Grab Situation to use the new Character Grab System.

    - In game inspector update:
        - Modular scene system with assets types. (Characters, Scene Meshes, Scene Background, Scene Lights)

    - Full refactoring of the IK system.
        Used for retarget character animations, Exemple: 
            - Talas stomps offset.
            - VR body animations.
            - or Keep Cobble paws one the floor.
        - Modular with context-specific embedded code in modules.

    - Small refactoring with wrinkles animations and shaders.
        - More modular, can be used on all characters and it now fully controlled by the animation script.

    - VR Fullbody Tracking:
        - New Control rig to manage character animtions with tracked points.
        - Full refactoring of the VR fingers tracking and netcode replication.

    - New Cheat Manager:
        New in-game cheat manager with per-cheat details, console comments and conditional uses.

    - New debug tools:
        - In editor nodal actor to manage tool logics.
        - New debug widget and properties showing.

    - Stuck System, Light Refactoring:
        - New config based procedural Stuck Manager.
        - Support of VR profiles.

    - General progress in side tools used by MMVS (BPS, BBPL, UE AE, ...)

## Bugs
    - Fixed: Moving animations when crouch is too fast when very small.
    - Fixed: With extreme character scales, the near render clip cut the character.
    - Fixed: The character use walk animation than slide when pushed.
    - Many bugs fix but I did not noted all...
